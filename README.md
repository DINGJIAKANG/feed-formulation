# 线性规划饲料配方优化系统

一个基于 Streamlit 的饲料配方优化工具，使用线性规划算法自动计算最低成本的饲料配方。

## 功能特点

- ✅ 支持猪、禽、牛、羊、其他（猫/狗/鱼/马）等多种动物
- ✅ 自动从 548 种原料数据库中匹配最佳配方
- ✅ 支持上传自定义原料数据和饲养标准
- ✅ 高级约束：营养上限、精粗比、钙磷比
- ✅ 配方微调功能，支持手动调整配比
- ✅ 干物质基础（DM）计算
- ✅ 自动排除非蛋白氮（NPN）原料（单胃动物）
- ✅ 特殊原料默认用量上限（避免过度使用廉价非常规原料）

## 在线使用

部署后访问：`https://your-app-name.streamlit.app`

## 本地运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动应用

```bash
streamlit run streamlit_app.py
```

应用将自动在浏览器中打开：`http://localhost:8501`

## 使用方法

### 基本流程

1. **选择动物种类**（猪/禽/牛/羊/其他）
2. **选择饲养标准**（如"猪-生长肥育猪-20-50kg"）
3. **选择原料数据来源**：
   - 使用默认原料库（548 种原料）
   - 上传自定义原料数据
4. **设置配方要求**（ optional）：
   - 营养需求（下限/上限）
   - 原料用量限制
   - 精粗比约束
   - 钙磷比约束
5. **点击"开始求解"**
6. **查看结果**：配方详情、营养成分、成本分析

### 高级功能

- **配方微调**：在结果页面点击"配方微调"，手动调整配比
- **投料单转换**：将干物质基础（DM）配方转换为原样基础（as-fed）投料单
- **自动补充原料**：当从上传原料无法求解时，自动从默认库补充原料

- **openpyxl** - Excel 文件读写

## 算法说明

本程序使用**线性规划（Linear Programming）**求解最低成本配方：

- **目标函数**：最小化总成本 `min Σ(c_i * x_i)`
- **约束条件**：
  1. 总量约束：`Σx_i = 100%`
  2. 营养下限：`Σ(a_ni * x_i) >= 需求下限`
  3. 营养上限：`Σ(a_ni * x_i) <= 需求上限`
  4. 原料用量限制：`lb_i <= x_i <= ub_i`
  5. 精粗比约束：`concentrate_min <= Σx_i(精料) <= concentrate_max`
  6. 钙磷比约束：`ca_p_min <= Ca/P <= ca_p_max`
