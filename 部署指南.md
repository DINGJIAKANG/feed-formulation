# 饲料配方优化系统 - 部署指南

## 步骤 1：创建 GitHub 仓库

1. 访问 https://github.com/ 并登录
2. 点击右上角 "+" → "New repository"
3. 填写信息：
   - Repository name: `feed-formulation-app`
   - Description: `线性规划饲料配方优化系统`
   - 选择 "Public"
   - 勾选 "Add a README file" ❌（不要勾选，我们已经有 README.md）
4. 点击 "Create repository"

## 步骤 2：上传代码

### 方法 A：使用 Git 命令（推荐）

1. 打开命令提示符（`Win + R` → `cmd` → 回车）
2. 复制粘贴以下命令：

```cmd
cd /d "D:\python设计饲料配方\部署版"

git init
git add .
git commit -m "初始提交：饲料配方优化系统 v2.0"

git remote add origin https://github.com/你的用户名/feed-formulation-app.git
git push -u origin main
```

3. 输入 GitHub 用户名和密码（或 Token）

### 方法 B：使用 GitHub Desktop（更简单）

1. 下载并安装 GitHub Desktop：https://desktop.github.com/
2. 登录 GitHub 账号
3. 点击 "Add an Existing Repository from your Hard Drive"
4. 选择 `D:\python设计饲料配方\部署版` 文件夹
5. 点击 "Publish repository"
6. 填写仓库名：`feed-formulation-app`
7. 点击 "Publish Repository"

## 步骤 3：部署到 Streamlit Community Cloud

1. 访问 https://share.streamlit.io/
2. 点击 "Sign in with GitHub"
3. 授权 Streamlit 访问你的 GitHub 账号
4. 点击 "New app"
5. 填写信息：
   - **Repository**：选择 `feed-formulation-app`
   - **Branch**：`main`
   - **Main file path**：`streamlit_app.py`
   - **App URL**：自定义一个好记的地址（如 `feed-formulation-app`）
6. 点击 "Deploy!"

## 步骤 4：访问应用

部署完成后，你会得到一个公开 URL：

```
https://feed-formulation-app.streamlit.app
```

任何人都可以访问这个 URL 使用你的程序！

## 注意事项

1. **免费版限制**：
   - 应用闲置后会进入睡眠状态（下次访问需要等待 10-30 秒唤醒）
   - 每月有流量限制（但对个人项目足够）

2. **代码更新**：
   - 每次更新代码并推送到 GitHub 后，Streamlit Cloud 会自动重新部署
   - 通常 2-3 分钟即可完成更新

3. **文件大小限制**：
   - Streamlit Cloud 免费版有 200 MB 文件大小限制
   - 我们的程序只有 1.1 MB，远小于限制 ✅

## 常见问题

### Q1：部署失败怎么办？

检查 `requirements.txt` 是否完整，确保所有依赖都已列出。

### Q2：如何修改应用名称？

在 Streamlit Community Cloud 的 app 设置中，可以自定义 URL。

### Q3：如何添加自定义域名？

需要升级到付费版（Streamlit for Teams）。
